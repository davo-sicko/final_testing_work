Пройти вручную через `python menu.py`, отметить результат.

| № | Шаги | Ожидаемый результат | Статус |
|---|------|----------------------|--------|
| 1 | Запустить меню, выбрать "Показать все книги" на пустой базе | Сообщение "Список пуст." | ✅ |
| 2 | Добавить книгу (название, автор, год) | Книга добавлена, выведен id=1 | ✅ |
| 3 | Показать все книги | Книга отображается со статусом "не прочитано" | ✅ |
| 4 | Добавить книгу с пустым названием | Ошибка валидации, книга не добавлена | ✅ |
| 5 | Добавить книгу с годом "abc" (не число) | Сообщение "год должен быть числом" | ⬜ |
| 6 | Добавить книгу с отрицательным годом | Ошибка валидации | ✅|
| 7 | Найти книгу по части названия (другой регистр) | Книга найдена | ✅ |
| 8 | Найти книгу по автору | Книга найдена | ✅ |
| 9 | Поиск несуществующей книги | "Ничего не найдено." | ✅ |
| 10 | Отметить книгу прочитанной по id | Статус сменился на "прочитано" | ✅ |
| 11 | Отметить прочитанной книгу с несуществующим id | Сообщение об ошибке, без падения программы | ✅ |
| 12 | Отредактировать название книги | Название обновилось, остальные поля не изменились | ✅ |
| 13 | Удалить книгу по id | Книга удалена, не отображается в списке | ✅ |
| 14 | Удалить книгу с несуществующим id | Сообщение об ошибке | ✅ |
| 15 | Фильтр по статусу "read" при наличии прочитанных и непрочитанных | Показаны только прочитанные | ✅ |
